## Setup

1. Clone the project

```bash
    git clone https://github.com/chooselife-gdscuoj/choose-life
```

2. Navigate to the project

```bash
    cd choose-life
```

3. install dependencies

```bash
    npn install
```
4. Run project locally
  ```bash
      npn npm start
  ```
